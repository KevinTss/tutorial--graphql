# Tutorial - Graphql

## Setup yarn

Started by setup this repo as monorepo using yarn workspace

```bash
yarn init -w
```

Make sure workspace are initialized

```bash
yarn config set workspaces-experimental true
```

Checked all packages

```bash
yarn workspace list --json
```
